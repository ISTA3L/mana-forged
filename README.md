# Mana forged SRD
This repository contains the System Reference Document (SRD) for Mana Forged, a Forged in the Dark (FitD) game designed for fantasy settings. It includes the core mechanics and tools needed to play or build upon the system.

If you are looking for the complete version of the game, you can find it on DriveThruRPG and Itch.io.

Contributions are welcome! If you notice any spelling, formatting, or layout errors, feel free to report them or suggest improvements.

Please note that this project is developed without funding, so any support or feedback is greatly appreciated.
